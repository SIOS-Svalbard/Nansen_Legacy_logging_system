Label printers troubleshooting

Check through these things in order:


  1. Are the rollers clean?
    Check that the labels are passing through the printer smoothly. Carefully cut away any blocked paper.

  2. Are there any labels left?
    Replace the labels, following the pictures inside the printer. Press the 'feed' button (the only button on the top) after closing. Make sure the light is green. If not, open and close again.

  3. Is there any printer ribbon left?
    Replace the printer ribbon following the pictures inside the back of the printer. Make sure to use the correct printer ribbon!
    {{ribbon}} for {{size}} labels.
    Make sure the light is green. If not, open and close again.

  4. Is the label printing correctly?
    Factory reset the printer. Hold the 'feed' button (the only button on the top) down. The light will blink first once, then twice etc. Release after it blinks 4 times.

  5. Turn it off and on again
    Using the button at the back of the printer

Still no luck? Ask the person responsible for the printers.
